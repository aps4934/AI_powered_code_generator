🚀 AI Code Generator
A simple AI-powered code generation backend built with FastAPI and the OpenAI API.
It takes a programming prompt and language as input, then returns clean, functional code generated by AI.

Generate code in multiple programming languages

Uses OpenAI GPT models

Simple REST API for integration with any frontend

CORS enabled for easy local development
